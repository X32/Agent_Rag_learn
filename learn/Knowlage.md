结合这份**大模型Agent智能体开发实战冲刺班**的完整目录，该课程是一套**从环境搭建到企业级落地**的全栈式Agent开发课程，知识点覆盖**大模型基础、微调、RAG、Agent框架、实战项目**五大核心板块。以下是分模块的知识点总结：

### 一、 基础预备与环境搭建

**核心目标**：构建大模型开发的软硬件基础环境，掌握必备工具与语言能力。

| 知识点模块             | 核心内容                                                                                          |
| :--------------------- | :------------------------------------------------------------------------------------------------ |
| **编程基础**     | 大模型必备Python基础（31小节）：Python语法、数据处理、基础编程能力，为代码实战打基础。            |
| **深度学习基础** | 深度学习Transformer基础（16小节）：Transformer架构原理、Encoder/Decoder结构，理解大模型底层核心。 |
| **环境准备**     | 大模型部署软硬件环境准备（14小节）：本地/云端服务器配置、GPU选型、依赖库安装、环境隔离。          |
| **部署与调用**   | 大模型部署与调用方法（106小节）：大模型本地部署、API调用、不同模型（开源/闭源）的调用方案。       |
| **部署工具**     | 大模型部署策略与工具详解（13小节）：主流部署工具（如vLLM、TensorRT-LLM等）、部署优化策略。        |

### 二、 大模型微调全流程

**核心目标**：掌握大模型定制化能力，通过微调让模型适配特定场景与数据。

| 知识点模块             | 核心内容                                                                                           |
| :--------------------- | :------------------------------------------------------------------------------------------------- |
| **微调原理**     | 大模型高效&强化微调原理（38小节）：LoRA、QLoRA、全参数微调等高效微调技术原理，强化学习（RL）基础。 |
| **数据集构建**   | 微调数据集获取方法及自主构建（10小节）：公开数据集筛选、自定义数据集标注、数据清洗与预处理。       |
| **实战微调工具** | 借助Llama-Factory进行高效微调（30小节）：Llama-Factory工具使用、一键微调、参数配置、模型导出。     |
| **微调应用**     | 大模型微调&蒸馏应用实战（48小节）：微调模型的部署、应用开发；模型蒸馏（Distillation）原理与实践。  |
| **企业级微调**   | 企业级大模型微调实战（28小节）：企业场景下的微调方案、多任务微调、微调效果评估与优化。             |
| **冲刺专享**     | GPT-5/GPT-OSS微调实战公开课（28小节）：前沿GPT类模型微调方法，拓展前沿技术视野。                   |

### 三、 RAG检索增强技术

**核心目标**：解决大模型“知识过时”与“事实性”问题，掌握基于检索的知识增强方案。

| 知识点模块         | 核心内容                                                                                                                      |
| :----------------- | :---------------------------------------------------------------------------------------------------------------------------- |
| **RAG基础**  | RAG基础 & 向量数据库应用（20小节）：RAG核心流程（检索-生成）、向量数据库（Milvus、FAISS等）、文档切片与向量化。               |
| **RAG优化**  | 企业级RAG技术优化 & 评估（11小节）：RAG流程优化（召回优化、重排序、生成优化）、效果评估指标与方法。                           |
| **GraphRAG** | GraphRAG应用实战（46小节）：基于知识图谱的RAG架构、实体关系抽取、图检索与生成结合、企业知识库GraphRAG落地。                   |
| **加餐拓展** | RAG/GraphRAG/多模态RAG公开课（44小节）、PaddleOCR-VL多模态RAG系统实战（9小节）：多模态RAG（图片/OCR+文本）、复杂场景RAG优化。 |

| 知识点模块              | 核心内容                                                                                                                                                                                                                                                                                          |
| :---------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Agent基础**     | AI Agent基础与ReAct实战（15小节）：`<br>`1. Agent与LLMs+RAG的本质区别、Agent爆发原因、经典架构（AgentExecutor）。`<br>`2. **Function Calling**：完整生命周期、并行/多函数调用、智能客服场景落地。`<br>`3. **ReAct框架**：原理、从零复现、电商智能客服项目实战（当前学习进度）。 |
| **主流Agent框架** | 1. 自主框架：Langraph（37小节）：Langraph核心概念、工作流设计、Agent状态管理与实战。`<br>`2. 热门框架：AutoGen（8小节）、AssistantAPI & Swarm（23小节）、Agents SDK（17小节）、Google ADK（47小节）：各框架核心特性、开发流程、多智能体协作。                                                   |
| **低代码开发**    | 低代码开发 Dify/Coze/n8n（63小节）：低代码Agent开发平台使用、可视化搭建、快速部署与定制化。`<br>`加餐：Dify开发实战（23小节）：Dify平台深度应用、RAG+Agent结合开发。                                                                                                                            |

### 课程整体特点

1. **全链路覆盖**：从基础环境到企业落地，从技术原理到实战项目，无明显短板。
2. **技术前沿**：紧跟ReAct、GraphRAG、LangGraph、Agents SDK等最新技术与框架。
3. **实战导向**：4大核心实战项目+多个加餐实战，强调“学完即能用”。
4. **体系完整**：兼顾零基础入门（Python/Transformer）与高阶进阶（企业级微调、自定义Agent开发）。
