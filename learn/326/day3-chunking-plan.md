## 核心问题

### 1. 为什么要切分？不切分行不行？

- [X] 理解 LLM 的 context window 限制
- [X] 理解检索精度与文档长度的关系
- [X] 了解长文档带来的噪声问题

### 2. chunk_size 多大合适？为什么？

- [X] 理解 chunk_size 对检索精度的影响
- [X] 理解 chunk_size 对上下文完整性的影响
- [X] 了解不同场景的推荐值

### 3. chunk_overlap 是什么？为什么需要？

- [X] 理解 overlap 的作用
- [X] 掌握 overlap 与 chunk_size 的关系
- [X] 了解推荐的 overlap 比例

### 4. 有哪些切分策略？

| 策略         | 特点                         | 适用场景           |
| ------------ | ---------------------------- | ------------------ |
| 固定长度切分 | 简单直接，按字符数切         | 通用场景           |
| 句子切分     | 按自然句子边界切             | 需要语义完整的场景 |
| 递归切分     | 先按段落，再按句子，层层递归 | 推荐的默认策略     |
| 语义切分     | 根据内容语义相似度切分       | 高级场景           |
