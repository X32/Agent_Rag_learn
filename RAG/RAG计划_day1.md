写一个最简单的 RAG 程序：

1. 读取一个 TXT 文件
2. 把文本切分成小块
3. 用 OpenAI API 把文本块转成向量
4. 用户提问时，找到最相似的文本块
5. 把文本块和问题一起发给 LLM，生成答案

要求：代码尽可能简单，每行都要有注释
